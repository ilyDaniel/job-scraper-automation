# Job Scraper (Work in Progress)

This project is a simple Python-based job scraper that is currently under development.

The goal is to build a tool that:

* Scrapes job listings from company career pages
* Filters jobs based on keywords (e.g. AI, Software Engineer)
* Stores results in a file or database
* Sends notifications when new relevant jobs are found

This is a work in progress and features are being added incrementally.
